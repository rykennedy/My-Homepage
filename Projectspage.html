Eye-Movement Demo

This project is a demonstration in responsive design using HTML, CSS, and Javascript. Simply move your mouse across the screen and watch the eyes follow!

PacMen-a-palooza!

This is an exercise showcasing mastery over basic DOM principles and techniques using Javascript. Click the "Add PacMan" button to generate as many PacMen as you want and click "Start Game" to watch watch them bounce across the screen.

Future iterations: add buttons to randomize the pacman color and stop the game.

Live Bus Tracker

This is one of my favorite WIPs. This page utilizes asynchronous code in conjuction with Mapbox GL and the Whatcom Transportation Authority's public API to track bus locations in real time.
